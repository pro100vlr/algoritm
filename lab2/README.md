## Запуск проекта 

1. Клонируйте репозиторий:
  
`git clone https://github.com/pro100vlr/algoritm.git`  

2. Перейдите в папку с проектом:

`cd algoritm/lab2/`  

3. Запуск всех задач:
   
`bash ../run_all_one_lab.sh`

4. Запуск всех тестов задач:
   
`bash ../run_all_tests_one_lab.sh`
